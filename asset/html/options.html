When you click on the extension icon, proxy mode will cycle in the following modes:

  • DIRECT - no proxy
  • PAC - use PAC file
  • TUNNEL - all traffic are go through ssh tunnel

if PAC file url is not set, then pac mode is skipped automatically.
if ssh tunnel host and port is not set, then tunnel mode is skipped automatically.

for example:
  • file:///Users/oylbin/proxy.pac
  • http://ouyang.me/proxy.pac

, for example: 127.0.0.1
, for example: 8527
